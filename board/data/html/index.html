🌐 Online Mode

Connect to WiFi and send data to the cloud in real-time. Requires internet connection and user account.

📴 Offline Mode

Store readings locally on the device. No WiFi or internet required. Perfect for remote locations.

  • Stores up to ~10,000 readings (~1 day at 10s interval)
  • Data can be synced later when online
  • Can exit offline mode anytime
